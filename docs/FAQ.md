## 🤔 כללי

### מה זה HollyRap?
**HollyRap** הוא מאגר מקיף להפיכת באטל ראפ לכלי של קדושה והעצמה. המטרה היא ליצור אלטרנטיבה חיובית ועוצמתית לתרבות הראפ המסורתית.

### מי יכול להשתמש במאגר?
כל אחד! המאגר פתוח לכולם - יוצרים צעירים, מחנכים, קהילות, וכל מי שמעוניין להשתמש בכוח המילים לטובה.

### האם יש עלות לשימוש?
לא! המאגר הוא פתוח וחינמי לשימוש.

### מה הרישיון?
המאגר מורשה תחת רישיון MIT - ראה [LICENSE](../LICENSE) לפרטים.

## ✍️ כתיבה

### איך מתחילים לכתוב באטל?
1. קרא את [מדריך ההתחלה המהירה](GETTING_STARTED.md)
2. עיין ב[מדריך הכתיבה](STYLE_GUIDE.md)
3. קרא [באטלים לדוגמה](../content/battles/)
4. תרגל כתיבת באטל ראשון

### כמה שורות צריך באטל?
באופן כללי, 12-16 שורות לכל משתתף. זה מספיק להעביר מסר מלא וחזק.

### איך בודקים חריזה?
חריזה היא כשהסופים של שורות נשמעים דומה. לדוגמה:
- שורה א מסתיימת ב-"ה"
- שורה ב מסתיימת ב-"ה"
- זו חריזה!

### מה אם אני לא יודע עברית טוב?
זה בסדר! אתה יכול:
- להשתמש במילון
- לבקש עזרה מהקהילה
- לכתוב ברמה שלך ולשפר בהדרגה

### האם צריך לצטט מקורות תורניים?
לא חובה, אבל מומלץ מאוד! ציטוטים תורניים מוסיפים עומק ומשמעות לבאטל.

### מה אם המסר שלי לא תורני מספיק?
זה בסדר! המטרה היא מסר חיובי וערכי. תורניות היא בונוס, לא דרישה.

## 🤝 תרומה

### איך אני יכול לתרום?
ראה את [מדריך התרומה](../CONTRIBUTING.md) לפרטים מפורטים. אתה יכול:
- להוסיף באטלים חדשים
- לשפר תיעוד
- לדווח על באגים
- להציע תכונות חדשות

### איך מוסיפים באטל חדש?
1. כתוב את הבאטל בקובץ Markdown
2. שמור ב-`content/battles/`
3. פתח Pull Request
4. עקוב אחר התבנית

### האם אני צריך אישור לפני תרומה?
לא! פשוט פתח Pull Request ואנו נבחן אותו.

### כמה זמן לוקח לבחינת Pull Request?
בדרך כלל 1-3 ימים. אם יש דחיפות, ציין זאת ב-PR.

### איך מתקנים שגיאות כתיב?
פשוט פתח Pull Request עם התיקון. כל תרומה קטנה עושה הבדל!

## 🛠️ טכני

### איך מעתיקים את המאגר?
```bash
git clone https://github.com/AnLoMinus/hollyrap.git
```

### איך מתקינים את המאגר?
אין צורך בהתקנה! זה מאגר של קבצים טקסטואליים. פשוט clone והוסף עורך טקסט.

### איזה עורך טקסט מומלץ?
כל עורך טקסט עובד! מומלץ:
- VS Code
- Obsidian
- Sublime Text
- או כל עורך שאתה נוח איתו

### איך מעדכנים את המאגר?
```bash
git pull origin main
```

### איך יוצרים branch חדש?
```bash
git checkout -b feature/amazing-feature
```
