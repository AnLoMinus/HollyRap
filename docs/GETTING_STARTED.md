# 🚀 התחלה מהירה

מדריך זה יעזור לך להתחיל עם HollyRap במהירות.

## 📌 מה זה HollyRap?

**HollyRap** הוא מאגר מקיף להפיכת באטל ראפ לכלי של קדושה והעצמה. המטרה היא ליצור אלטרנטיבה חיובית ועוצמתית לתרבות הראפ המסורתית.

## 🎯 צעדים ראשונים

### 1️⃣ הכרות עם המאגר

1. **קרא את ה-README** - [README.md](../README.md)
2. **עיין בדוגמאות** - [באטלים לדוגמה](../content/battles/)
3. **למד את הטכניקות** - [מדריך כתיבה](STYLE_GUIDE.md)

### 2️⃣ הבן את המבנה

```
hollyrap/
├── content/          # תוכן הבאטלים
│   ├── battles/      # קבצי באטלים
│   └── guides/        # מדריכים
├── docs/             # תיעוד
├── website/          # קבצי האתר
└── assets/           # תמונות
```

### 3️⃣ התחל לכתוב

1. **קרא באטלים קיימים** - הבן את הפורמט
2. **למד טכניקות** - עיין ב[מדריך כתיבה](STYLE_GUIDE.md)
3. **כתוב באטל ראשון** - התנסה בכתיבה
4. **שתף עם הקהילה** - פתח Pull Request
