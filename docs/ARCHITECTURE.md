# 🏗️ מבנה הפרויקט

תיעוד זה מתאר את הארכיטקטורה והמבנה של פרויקט HollyRap.

## 📁 מבנה תיקיות

```
hollyrap/
├── .github/                  # הגדרות GitHub
│   ├── workflows/            # CI/CD workflows
│   ├── ISSUE_TEMPLATE/       # תבניות Issues
│   └── PULL_REQUEST_TEMPLATE.md
├── content/                  # תוכן הבאטלים
│   ├── battles/              # קבצי באטלים (Markdown)
│   └── guides/               # מדריכים כתיבה
├── docs/                     # תיעוד
│   ├── INDEX.md              # שער ניווט
│   ├── GETTING_STARTED.md    # התחלה מהירה
│   ├── ARCHITECTURE.md       # קובץ זה
│   ├── STYLE_GUIDE.md        # מדריך כתיבה
│   └── ...
├── website/                  # קבצי האתר
│   ├── index.html
│   ├── battles-archive.html
│   ├── styles.css
│   ├── scripts.js
│   └── battle-content.js
├── assets/                   # משאבים
│   └── images/               # תמונות
├── README.md                 # קובץ ראשי
├── LICENSE                   # רישיון MIT
├── CHANGELOG.md              # היסטוריית שינויים
├── CONTRIBUTING.md           # מדריך תרומה
├── SECURITY.md               # מדיניות אבטחה
├── CODE_OF_CONDUCT.md        # קוד התנהגות
├── ROADMAP.md                # מפת דרכים
├── .gitignore                # קבצים להתעלמות
├── .editorconfig             # הגדרות עורך
└── CODEOWNERS                # בעלי זכויות
```

## 📄 קבצים עיקריים

### מסמכי בסיס
- **README.md** - תיאור הפרויקט ומדריך מהיר
- **LICENSE** - רישיון MIT
- **CHANGELOG.md** - היסטוריית שינויים
- **CONTRIBUTING.md** - מדריך תרומה
- **SECURITY.md** - מדיניות אבטחה
- **CODE_OF_CONDUCT.md** - קוד התנהגות
- **ROADMAP.md** - מפת דרכים עתידית

### תיקיות תוכן
- **content/battles/** - קבצי באטלים בפורמט Markdown
- **content/guides/** - מדריכי כתיבה וטכניקות
- **docs/** - תיעוד מקיף של הפרויקט
- **website/** - קבצי HTML/CSS/JS לאתר
- **assets/** - תמונות וישויות

### תצורה
- **.gitignore** - קבצים להתעלמות מ-Git
- **.editorconfig** - הגדרות עורך אחידות
- **CODEOWNERS** - בעלי זכויות לקבצים
- **.github/** - הגדרות GitHub (workflows, templates)

## 🎤 מבנה באטל

באטל ראפ קדוש מורכב מ:

```
# שם הבאטל

## תיאור
[תיאור קצר של הנושא והמסר]

## המשתתפים
- **משתתף א'**: [שם/תפקיד]
- **משתתף ב'**: [שם/תפקיד]

## הבאטל

### משתתף א'
[12-16 שורות ראפ]

### משתתף ב'
[12-16 שורות ראפ]

## מסר ומוסר השכל
[מסר סיכום]

## מקורות וציטוטים
[ציטוטים תורניים רלוונטיים]
```

## 🌐 מבנה האתר

האתר מורכב מ:
- **index.html** - דף בית ראשי
- **battles-archive.html** - ארכיון באטלים
- **styles.css** - עיצוב CSS
- **scripts.js** - לוגיקה JavaScript
- **battle-content.js** - תוכן הבאטלים

## 🔄 תהליך עבודה

### עבודה עם Git
1. **Fork** את המאגר
2. **Clone** את ה-fork
3. **צור branch** חדש
4. **עשה שינויים**
5. **Commit** ו-**Push**
6. **פתח Pull Request**

### עבודה עם תוכן
1. **קרא באטלים קיימים**
2. **למד את הפורמט**
3. **כתוב באטל חדש**
4. **בדוק חריזה ומשקל**
5. **וודא מסר חיובי**
6. **פתח Pull Request**

## 📋 סטנדרטים

### קבצי Markdown
- קידוד: UTF-8
- סיום שורות: LF
- רוחב שורה: 80 תווים (אופציונלי)

### קבצי קוד
- עקוב אחר `.editorconfig`
- שמור על עקביות בפורמט
- הוסף הערות כשצריך
